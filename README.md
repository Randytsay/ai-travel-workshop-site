# AI 旅遊規劃師工作坊教學網站

這是一個為實體工作坊設計的互動式教學網站，目標是協助 AI 初學者在課堂中完成：

1. AI 3D 可愛公仔圖
2. AI 旅遊規劃需求整理
3. NotebookLM 研究接力
4. LINE 貼圖素材生成
5. 作品互動投票與成果牆
6. 課後講義領取導流
